## Features Implemented

- React + Tailwind CSS Setup
  Initialized using create-react-app
  Tailwind CSS configured for utility-first styling

- Product Data Fetching
  Integrated dummy API to fetch product data
  Created reusable hooks for get api data

- Product Listing Grid
  Responsive grid layout for displaying products
  add hover effect also on product card

- custom filtering
  Category filter using radio buttons
  Price filter using a range slider

- Search Functionality
  Search bar filters products by title and description
  Real-time data updation on searching value

- Product Details Page
  Clickable product cards redirected to detailed view
  Displays full product information

- Cart Sidebar
  Slide-in cart from the right
  Shows added items with quantity and price
